# deleteItem() method

Version added: `3.0.1`, `4.0.0`

Deletes item.  
**Note**: If you want an item to be deleted from the original array as well use two-way data binding syntax on `[(items)]` field.

## Parameters

`item: any`  
Item to delete.

## Returns

`Promise<void>`  
Promise that resolves when item has been deleted.
